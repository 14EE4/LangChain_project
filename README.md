# Email Draft Agent (LangChain + Groq)

대화형으로 이메일 초안을 작성하고, 톤 추천/수정/모의 발송까지 지원하는 CLI 에이전트입니다.

## 주요 기능

- 수신자/목적을 바탕으로 이메일 톤 추천 (공식/중립/친근)
- 이메일 초안 생성 (한국어)
- 초안 수정 요청 반영
- 최근 초안 조회
- 모의 발송 처리

## 프로젝트 구조

- app.py: CLI 실행 진입점
- agent.py: LangChain 에이전트 생성
- Tools.py: 톤 추천, 초안 생성/수정/발송 도구
- Models.py: Pydantic 데이터 모델
- mock_db.py: 메모리 기반 저장소

## 요구 사항

- Python 3.10+
- 가상환경 (.venv 권장)
- GROQ_API_KEY

## 설치

1) 가상환경 활성화

Windows PowerShell:

```powershell
& .\.venv\Scripts\Activate.ps1
```

2) 패키지 설치

```powershell
python -m pip install langchain langchain-core langchain-groq python-dotenv pydantic
```

3) 환경 변수 설정 (.env)

프로젝트 루트에 .env 파일을 만들고 아래처럼 입력합니다.

```dotenv
GROQ_API_KEY=your_groq_api_key
```

## 실행

```powershell
python .\app.py
```

실행 후 프롬프트에서 여러 턴으로 대화를 이어갈 수 있습니다.
종료 명령: exit, quit, q

## 입력 예시

```text
you> 이메일 초안을 작성해줘. 수신자: 김민수(kim@example.com), 목적: 프로젝트 일정 변경 안내
you> 보내는 사람은 홍길동이야
you> 좀 더 정중하게 바꿔줘
you> 발송해줘
you> exit
```

## 출력 예시 (프로그램 시작 화면)

프로그램 실행 시 아래와 같이 시작 화면이 표시됩니다:

```text
======================================================================
✉️  AI 이메일 드래프트 어시스턴트
======================================================================

이 프로그램은 Groq의 LLM을 활용하여 전문적인 이메일 초안을 빠르게 작성합니다.

기능:
  • AI가 자동으로 완성도 높은 이메일 초안 생성
  • 톤(공식/중립/친근) 추천 및 적용
  • 초안 수정 및 버전 관리
  • 메모리에 임시 저장 (프로세스 종료 시 초기화)

시작하기 - 다음 정보를 제공하면 초안을 만들 수 있습니다:

  [필수]
    • 수신자 이름          - 예: '김민수'
    • 수신자 이메일        - 예: 'info@example.com'
    • 목적                 - 예: '회의 일정 조율'
    • 보내는 사람 이름      - 예: '홍길동'

  [선택]
    • 핵심 포인트         - 세미콜론 ';'로 구분. 예: '빠른 응답 필요;주말 회피'
    • 톤                  - 공식 / 중립 / 친근 (언급하지 않으면 자동 추천)

예시: '홍길동(info@example.com)에게 이메일 초안을 만들어줄래? 회의 일정을 조율하는 내용으로'

종료: 'exit', 'quit', 또는 'q' 입력
======================================================================

you> 이메일 초안을 작성해줘. 수신자: 김민수(kim@example.com), 목적: 프로젝트 일정 변경 안내
you> 보내는 사람은 홍길동이야
you> 좀 더 정중하게 바꿔줘
you> 발송해줘
you> exit

agent> 추천드린 이메일 톤은 **중립**입니다.
agent> 이 톤으로 진행해도 괜찮으신가요? 필요하시면 다른 톤으로 바꾸어 드릴 수 있습니다.

agent> **초안**
제목: [프로젝트 일정 변경 안내] 관련 문의 드립니다

김민수님께,

프로젝트 일정 변경 안내 관련하여 메일드립니다.

핵심 내용은 아래와 같습니다.
- 프로젝트 일정 변경
- 새로운 일정 및 마일스톤
- 회의 일정 조정

추가로 필요한 사항이 있으면 말씀 부탁드립니다.

검토 부탁드립니다.
홍길동 드림

agent> 더 정중한 톤(공식)으로 수정했습니다.
agent> 모의 발송 완료: status=sent
종료합니다.
```

## 톤(Tone) 옵션

- **공식**: 외부 고객, 임원진, 긴급 상황에 적합
- **중립**: 일반적인 업무 커뮤니케이션에 적합
- **친근**: 감사, 축하, 관계 강화 목적에 적합

## 참고

- 이 프로젝트의 발송 기능은 실제 메일 전송이 아닌 모의 발송입니다.
- 모든 이메일은 한국어로만 생성됩니다.
- 초안은 프로세스 메모리에만 저장되며, 프로그램 종료 시 초기화됩니다.
